# 🌍 Dhaka to World

## 📌 Overview

A platform connecting Bangladesh’s rich heritage to the global audience by showcasing local culture and traditions.

## 🎯 Goal

To promote Bangladeshi heritage and bring local products to the international stage.

## 🛠️ Technologies Used

* HTML
* CSS

## ⚙️ Features

* Cultural storytelling
* Global exposure of local heritage
* Clean and informative UI

## 🚀 How to Run

Open `index.html` in your browser.

## 🌱 Future Improvements

* Add e-commerce functionality
* Multi-language support
* Global shipping integration
